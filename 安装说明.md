# 依赖安装说明

## 方法一：使用安装脚本（推荐）

### Windows PowerShell
在项目根目录下执行：
```powershell
.\install-dependencies.ps1
```

### Windows CMD
在项目根目录下执行：
```cmd
install-dependencies.bat
```

## 方法二：手动安装

### 1. 安装后端依赖

打开终端，进入后端目录：
```bash
cd backend
npm install
```

### 2. 安装前端依赖

打开终端，进入前端目录：
```bash
cd frontend
npm install
```

## 前置要求

1. **Node.js 已安装**
   - 检查是否已安装：在终端运行 `node --version`
   - 如果未安装，请访问 https://nodejs.org/ 下载安装

2. **npm 已安装**
   - 检查是否已安装：在终端运行 `npm --version`
   - npm 通常随 Node.js 一起安装

## 常见问题

### 问题1：npm 命令未找到
**解决方案：**
- 确保已安装 Node.js
- 重启终端或重新加载环境变量
- 检查 Node.js 是否在系统 PATH 中

### 问题2：路径中包含中文字符导致错误
**解决方案：**
- 使用安装脚本（install-dependencies.bat 或 .ps1）
- 或者手动在终端中切换到对应目录后执行 npm install

### 问题3：安装速度慢
**解决方案：**
- 使用国内镜像源：
  ```bash
  npm config set registry https://registry.npmmirror.com
  ```
- 或使用 cnpm：
  ```bash
  npm install -g cnpm --registry=https://registry.npmmirror.com
  cnpm install
  ```

## 安装完成后

1. **后端**：配置 `.env` 文件（参考 `backend/env.example`）
2. **数据库**：运行数据库迁移
   ```bash
   cd backend
   npm run prisma:generate
   npm run prisma:migrate
   ```

3. **启动项目**：
   - 后端：`cd backend && npm run dev`
   - 前端：`cd frontend && npm run dev`
